# Poppo
Poppo is a simple tweet library for Swift

## Prerequisites

#### Swift

- Swift 3.0 Release

#### macOS

- OS X El Captain 10.11.6

#### Linux

- Ubuntu 14.04 (not supported URLSessionHTTPClient, please use CurlHTTPClient for a while)

#### Other

- curl 7.35.0
